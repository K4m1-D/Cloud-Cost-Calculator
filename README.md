# Cloud Cost Calculator

A web application for estimating monthly cloud computing costs across AWS, Azure, and Google Cloud Platform.

## Live Demo

https://k4m1-d.github.io/Cloud-Cost-Calculator/

## Features

- Cost estimates for three major cloud providers
- Real time calculation
- Responsive design for mobile and desktop
- Input validation
- Clear cost breakdowns

## Technologies

- HTML5
- CSS3
- JavaScript ES6
- GitHub Actions for CI/CD
- GitHub Pages for hosting

## Local Development

1. Clone the repository
2. Open index.html in your browser
3. No build process required
